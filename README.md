## Cloud Log Monitoring System

# Overview
A Python-based log monitoring system that simulates application logs and detects abnormal error behavior.

# Features
- Log generation (INFO, WARNING, ERROR)
- Log analysis using Python
- Error threshold alert system
- Simple monitoring logic similar to cloud tools

# Technologies Used
- Python
- File handling
- Git & GitHub

# Learning Outcome
This project demonstrates basic DevOps monitoring and alerting concepts used in real-world systems.
